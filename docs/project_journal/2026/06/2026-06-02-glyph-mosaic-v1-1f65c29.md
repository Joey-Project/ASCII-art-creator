# Glyph Mosaic Creator V1

## Summary

- Implemented the Glyph Mosaic Creator V1 static SPA, docs, journal baseline, CI, GitHub Pages workflow, unit tests, and Playwright e2e.
- Addressed review hardening for source-pixel grid invalidation, async generation snapshots, uploaded/local font-name rendering safety, glyph renderability filtering, upload error recovery, raster export size caps, and stronger e2e assertions.
- Addressed final review hardening for source-pixels aspect preservation at grid bounds, font-size-aware glyph feature sampling, and grapheme-safe fallback segmentation when `Intl.Segmenter` is unavailable.
- Addressed final independent review findings by making ASCII opt-in controls authoritative over the default user glyph field and by blocking SVG export for non-embeddable Local Font Access fonts instead of silently producing a non-fidelity SVG.
- Addressed GitHub Codex findings by limiting generic fallback-signature glyph filtering so valid ASCII glyphs and authorized Local Font Access fonts are not dropped.
- Configured GitHub Pages for the custom domain `glyph-mosaic-creator.mahane.me` and root-path Vite assets.
- The app is browser-local and deployable as a static GitHub Pages build.

## Current State

- Product is a Vite + TypeScript + pnpm static SPA for GitHub Pages.
- Default glyph candidates are ASCII only.
- Non-ASCII glyphs are opt-in through user input or explicit glyph packs.
- Rendering uses a fixed cell grid where each cell chooses its own `glyph + font + weight + color`.
- Font support includes default presets, uploaded fonts, and Local Font Access as progressive enhancement.
- Grid controls include direct rows/columns and pixels-per-glyph calculation, with recommended defaults derived from uploaded image dimensions.
- Matching includes density prefiltering, local shape features, edge direction, local contrast, color modes, and optional dithering.
- Playwright e2e covers desktop/mobile layout, upload-to-preview, source-pixels grid settings, opt-in non-ASCII behavior, and all requested export formats.
- GitHub Pages is configured for `https://glyph-mosaic-creator.mahane.me/`.
